Code Sample: Pulling Metrics

# Sample: compute YoY growth
from dataclasses import dataclass

@dataclass
class Metric:
    current: float
    prior: float

    def yoy(self) -> float:
        return (self.current / self.prior - 1.0) * 100

rev = Metric(current=1250.0, prior=1096.0)
print(f"Revenue YoY: {rev.yoy():.2f}%")

Financial Formulas (MathJax)

Weighted Average Cost of Capital (WACC):

\[ \text{WACC} = \frac{E}{V} r_e + \frac{D}{V} r_d (1 - T) \]

Discounted Cash Flow (perpetuity with growth):

\[ \text{Value} = \frac{\text{FCF}_1}{r - g} \]

Earnings Per Share (EPS): \( \displaystyle EPS = \frac{\text{Net Income}}{\text{Weighted Avg. Shares}} \)

Explain each variable: E (equity), D (debt), V (E+D), r_e, r_d, tax T, growth g.
